# example-http-error418-FastAPI
 Making this proyect only to use the HTTP error 418, named "I'm a teapot". This example is made using FastAPI Python's framework

## Set up
Clone repository:

`git clone https://github.com/josecamporivas/example-http-error418-FastAPI`

`cd ./example-http-error418-FastAPI`

Install dependencies (you need Python3):

`pip install "fastapi[all]"`

Run local server:

`uvicorn main:app --reload`

Open web browser and type:

`http://localhost:8000/error418`

Enjoy the error code! :blue_heart:
